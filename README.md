# Cancer Mutation Benchmark Datasets

This repository stores benchmark datasets generated by the Panchenko Lab and used in multiple publications. These benchmarks compile experimentally validated cancer driver and neutral mutations from diverse sources and serve as reference datasets for evaluating mutation effect prediction algorithms.

### 2. **`cancer_mutations_benchmark.csv`** and **`STable_1.xlsx`** 

These datasets were used in:

> **Ostroverkhova, D., Sheng, Y., & Panchenko, A. (2024).**
> *Are next-generation pathogenicity predictors applicable to cancer?*
> *Journal of Molecular Biology, 168*, 168644.
> [https://doi.org/10.1016/j.jmb.2024.168644](https://doi.org/10.1016/j.jmb.2024.168644)

1. Martelotto, L.G. *et al.* **Benchmarking mutation effect prediction algorithms using functionally validated cancer-related missense mutations.** *Genome Biol* **15**, 484 (2014).
2. Olivier, M. *et al.* **The IARC TP53 database: new online mutation analysis and recommendations to users.** *Hum Mutat* **19**, 607â€“614 (2002).
3. Starita, L.M. *et al.* **Massively Parallel Functional Analysis of BRCA1 RING Domain Variants.** *Genetics* **200**, 413â€“422 (2015).
4. Mahmood, K. *et al.* **Variant effect prediction tools assessed using independent, functional assay-based datasets: implications for discovery and diagnostics.** *Hum Genomics* **11**, 10 (2017).
5. Campbell, B.B. *et al.* **Comprehensive Analysis of Hypermutation in Human Cancer.** *Cell* **171**, 1042â€“1056.e10 (2017).
6. Ng, P.K. *et al.* **Systematic Functional Annotation of Somatic Mutations in Cancer.** *Cancer Cell* **33**(3), 450â€“462 (2018).
7. Kim, E. *et al.* **Systematic Functional Interrogation of Rare Cancer Variants Identifies Oncogenic Alleles.** *Cancer Discov* **6**, 714â€“726 (2016).

Driver and passenger mutation labels were assigned according to:

> Brown, A.L., Li, M., Goncearenco, A., & Panchenko, A.R. (2019).
> *Finding driver mutations in cancer: elucidating the role of background mutational processes.*
> *PLoS Comput Biol*, 15(4), e1006981.
> [https://journals.plos.org/ploscompbiol/article?id=10.1371/journal.pcbi.1006981](https://journals.plos.org/ploscompbiol/article?id=10.1371/journal.pcbi.1006981)
